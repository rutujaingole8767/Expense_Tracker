## Project Description

Expense Tracker is a Python command-line application that helps users record and manage daily expenses. Users can add expenses, view all expenses, and calculate the total amount spent.

## Features

* Add Expenses
* View Expenses
* Calculate Total Expense
* Simple Command-Line Interface

## Technology Used

* Python

## How to Run

1. Install Python.
2. Open Command Prompt.
3. Run:
   python expense_tracker.py

## Conclusion

This project demonstrates Python concepts such as lists, loops, functions, user input, and basic calculations to manage personal expenses.
